Enable popup free-opening for Web Extensions

Firefox Setup Guide

About The Current Restriction

By default, your Firefox version prevents web extensions from opening their popups without a user gesture.

Why does this restriction exist?
This is a legacy measure initially introduced for ergonomic and security reasons. As Web Extension APIs have evolved, this restriction has become less relevant. In 2022, a new preference was added to Firefox that allows users to control this behavior while keeping it disabled.

Is this still relevant?
No, not really. Mozilla plans to disable this restriction in future Firefox versions to harmonize its default behavior with other browsers.

Why disabling this restriction can improve your user experience
The 2FAuth web extension offers to capture QR code from webpages to automatically register 2FA accounts to your 2FAuth database. With the restriction enabled, when you capture a QR code, you'll receive a notification prompting you to click the extension icon to open the popup and complete the registration process.

Without this restriction, the popup would open automatically without any notification, streamlining the process.

Can it be disabled safely?
Yes. This setting only affects the way extension popup opening is triggered. It does not affect the websites you visit or other security features.

Steps to Enable Automatic Popup Opening

1 Open Firefox Configuration

Type about:config in your Firefox address bar and press Enter.

2 Accept the Warning

Click the "Accept the Risk and Continue" button to access advanced settings.

3 Search for the Preference

In the search bar at the top of the page, type: extensions.openPopupWithoutUserGesture.enabled

4 Edit the Preference

Click the toggle button next to the preference name to change its value from false to true.

5 Restart Firefox (Optional)

While not always necessary, restarting Firefox ensures the change takes effect immediately.

All done!

The 2FAuth extension will now be able to open automatically when you capture a QR code.
